# Matcha
Matcha is a dating website, made as a student project for 42 school.

## Stack
- ReactJS
- Redux
- NodeJS (express)
- MySQL
- Bootstrap & MaterialUI (MDBReact)
- Socket.IO

## How to
Make sure to have MySQL started locally (or Update .env)
Update email informations in .env file (!Important!)
```
npm install
```
```
npm run dev
```
